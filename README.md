## 1. 주제 선정 이유
**새로운 사람을 만나고 싶은 당신, moim에서 찾아보세요?
**
```
새로운 인연을 맺기 어려운 사람과 특정 활동에 동참할 사람을 찾기 어렵고 만남 주선에 부담이 있는 사람을 위한 서비스
```
